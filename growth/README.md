# growth/

Draft outputs from the `team-growth` agent (DevRel / growth strategist).
Everything here is a **draft for Pelayo to review** — nothing is ever posted
automatically. Posting to LinkedIn / X / anywhere is a manual action.

## Structure
- `ideas/YYYY-MM-DD-<topic>.md` — researched ideas from peer portfolios &
  dev-tool launches, each with a source URL and how it could apply here.
- `posts/YYYY-MM-DD-<topic>.md` — social post drafts, grouped by platform,
  promoting the connected body of work: the generative-semantic-ui library,
  Honest-DD, and Pelayo's career.

## How it gets filled
- Interactively: `/team` Phase 5 may spawn `team-growth` when the objective is
  about positioning or promotion.
- On schedule: the weekly `growth-digest` GitHub Action
  (`.github/workflows/growth-digest.yml`) opens a PR with fresh drafts here.

These tools improve together — the portfolio, the library, and HDD are one
story.
