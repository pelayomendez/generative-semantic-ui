I am Pelayo Méndez, a software developer operating at the intersection of aesthetic precision and generative logic. My work focuses on the transition from static interfaces to dynamic, intelligent environments that anticipate user intent and evolve in real-time.

I believe in Generative UI: the concept that a user interface should not be a fixed grid, but a fluid response to data. By leveraging LLMs and advanced pattern recognition, we can build tools that feel less like software and more like an extension of human thought.

In my lab, code is more than logic—it is a medium for sculptural expression. Every pixel is an architectural decision, every function a component of a larger, living system.
